# 怎么使用chrome的开发者工具分析页面性能

最近在分析自己的线上web产品，发觉chrome我最常用的就是network了，其他的也没有怎么用过。这里就研究一番。

# NetWork

这个标签还是最重要的，页面的所有请求都在这个面板有记录。

在面板的选择部分，有两个选项很好用
![20200628223157](http://tuchuang.funaio.cn/md/20200628223157.png)

这里的filter不仅可以使用关键字来过滤出需要的请求。也可以用关键字过滤掉不需要的请求。这点技巧可能不是所有人都知道。
```
config // 获取请求url中有config的
- config // 将请求url中有config的过滤掉
```

我们的网站现在大都在wifi环境中使用，但是有的时候，在网络不好的情况，这里就模拟了3G，Slow 3G，甚至你也可以自己定义一个网络环境（自己定义上行和下行限制）来看这种环境下页面的表现。
